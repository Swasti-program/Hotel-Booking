# 🏨 Hotel Booking page

A responsive Hotel Booking Website built using **HTML, CSS, and JavaScript**. This project provides a modern and user-friendly interface for browsing hotels, viewing room details, selecting check-in and check-out dates, and booking rooms.

## ✨ Features

- Responsive design for all devices
- Beautiful and modern UI
- Hotel listings with room details
- Check-in & Check-out date selection
- Booking form
- User-friendly interface
- Smooth navigation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Hotel-Booking/
│── index.html
│── style.css
│── script.js
│── images/
└── README.md
```

## 🚀 How to Run

1. Clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.
